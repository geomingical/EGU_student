Topic 1

素材輸入與草稿生成

Material Input & Script Generation

1.1 準備您的講稿

在開始口說練習之前，我們需要準備一份講稿。您不需要自己寫出完美的英文。只需將您的中文筆記、想法或經歷內容保存為 DOCX 或 PDF 檔案，然後直接上傳到 web ChatGPT 讓 AI 解析，並使用下方指令讓 AI 幫您生成自然且專業的英文口說草稿，同時為您整理出核心單字表。

Prompt 1: Script Generation
I want to practice speaking about a specific topic. I've uploaded my notes as a document. Please extract the key information from the document and generate a natural, professional spoken English script for me to practice. Keep the vocabulary accessible but professional. Also, provide a list of 5-8 key vocabulary words or phrases used in the script with their Chinese definitions and pronunciation guides.

使用提示：請先在 web ChatGPT 中上傳您的 DOCX 或 PDF 檔案，再將上方 Prompt 複製並貼上，AI 即可自動解析您的文件並生成講稿。
