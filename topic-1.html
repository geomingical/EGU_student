Topic 1

素材輸入與草稿生成

Material Input & Script Generation

1.1 準備您的講稿

在開始口說練習之前，需要準備一份講稿。您不需要自己寫出完美的英文。只需將您的研究內容的 DOCX 或 PDF 檔案，然後直接上傳到 web ChatGPT 讓 AI 解析，並使用下方指令讓 AI 幫您生成自然且專業的英文口說草稿，同時為您整理出核心單字表。

Prompt 1: Script Generation
You are helping a student prepare a short presentation for the EGU General Assembly. The student will either give an 8 minute oral presentation or a 3 minute poster pitch. They will upload a research file in PDF or DOCX format that includes a title, brief content, and figures. Your task is to generate a clear and simple speech script based only on the uploaded material. Use straightforward English and short sentences because the student speaks slowly. Avoid complex words and long explanations. The first sentence must clearly state the main purpose of the research. Then organize the script in a logical flow: 1.A short introduction to the problem or context 2.The method or approach used 3.The main results 4. The key takeaway or implication If figures are provided, briefly describe them in a way that is easy to say aloud. Keep the tone natural, like a human scientist explaining their work. Avoid exaggeration. Keep the script concise and suitable for speaking within the time limit.

使用提示：請先在 web ChatGPT 中上傳您的 DOCX 或 PDF 檔案，再將上方 Prompt 複製並貼上，AI 即可自動解析您的文件並生成講稿。然而，生成的講稿要在審閱編修
